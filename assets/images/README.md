# Assets Folder

Place your app images here:

- **app_logo.png** - App logo/icon (recommended size: 512x512 or larger)
- **splash_logo.png** - Logo for splash screen (recommended size: 512x512 or larger)

You can use any image format supported by Flutter (PNG, JPG, SVG, etc.).
PNG with transparency is recommended for the logo.

Suggested sizes:
- App Logo: 512x512px or 1024x1024px
- Splash Logo: 512x512px or larger
